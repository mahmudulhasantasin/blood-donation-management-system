# Blood Donation Management System

A console-based Blood Donation Management System written in C.

## Features
- Donor management using Linked List
- Donation history using Stack
- Patient request handling using Queue
- Search donors
- Save/load data from file
- Statistics dashboard

## Data Structures Used
- Linked List
- Stack
- Queue

## Language
C Programming

## How to Run

Compile:

gcc blood_donation_system_update.c -o blood.exe

Run:

./blood.exe
